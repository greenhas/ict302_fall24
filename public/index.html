Content Management System Defined As defined by Deane Barker, “A software system which automates the tasks around the management of content. A CMS assists editors in creating, organizing, controlling, securing, and ultimately delivering content.” (The Web Content Management Glossary, Barker 2016). My Understanding Content Management Systems or CMS’s are software that users can use in which help them deliver and organizer content. The software can be used to make creating a website or an app more efficient since a CMS would do certain parts of the process automatically allowing the user to focus on other things.
